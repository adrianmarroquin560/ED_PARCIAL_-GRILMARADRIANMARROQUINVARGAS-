Repositorio con las implementaciones Java y utilidades de benchmarking para el Parcial 1 de Estructura de Datos. Contiene las versiones iterativas y recursivas de Factorial, Fibonacci, Búsqueda Lineal y Burbuja, además de un medidor de tiempos que genera resultados/tiempos.csv. El objetivo es ejecutar los algoritmos, registrar 5 repeticiones por caso, y usar la plantilla Tiempos_Rendimiento.xlsx para analizar y comparar experimentalmente los Big‑O.
